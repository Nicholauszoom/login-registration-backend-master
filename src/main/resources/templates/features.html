Features

Enjoy Our-Service
Real-Time
Tracking
Real time tracking mode represents live data of tracked objects. Information is updated every time when visit a site. Monitored data contains vehicle status, latitude, longitude, altitude, address, speed, connection time, ignition status, fuel consumption, sensor data, nearest geozone and much more.
Wigets
Widgets display recent object information which is updated every ten seconds without need to refresh web page. Send commands to control device, view recent events and mileage graph.
DETAIL
REPORTS
Get detailed reports about trips, mileage, driving behavior, fuel usage and thefts, activity in particular zone or route. Reports are used for data analysis of particular vehicle or entire group. Reports can be exported or sent instantly to E mail address in HTML/PDF/XLS format.
Routes
Routes feature is a helpful tool to mark an important section of the road by drawing virtual path on the map. Additionally, get notifications if vehicle is within or out of route. This feature is useful to analyze vehicle dependency to the road.
Mobile Service
Android app with clear interface for GPS tracking for your devices and account management.
Dashboard/Chats
Dashboard is an information management tool that visually tracks, analyzes and displays important data of tracking account. Dashboard is efficient tool to monitor objects, events, maintenance, tasks and mileage in central location.
Proof of
Delivery
Employees can transfer electronic signatures using our mobile tracker app.

Receive delivery status updates in real-time
Task
Tasks make it easy to create and manage entries related to upcoming work which should be accomplished. Set start and end address, priority, task status. Tasks can be managed via GPS Tracker mobile application.
Monitor Driving
Behavior
Sensors help to expand GPS tracking capabilities. Read vehicle on-board computer data via CAN or get readings from externally connected sensors. Monitor different vehicle data, such as fuel level, RPM, temperature, voltage, iButton, RFID and more. Achieve precise measurements with calibration table and formulas.
History
History shows all stored data which server has collected from connected devices for chosen period of time. Software stores information received from GPS tracking devices, such as speed, time, location, stops, reports, events, etc. History is displayed in different ways: visually on the map, in a graph or HTML/XLS format.
POI & Tools
The Points of Interest (POI) tool enables you to add markers on locations, such as gas stations, hotels, restaurants, etc. You will also be able to name locations, add a description and use tools to calculate the distances between locations on the map
Expenses
Use the Expenses function to track the amount spent on an object maintenance. Evaluate the economic benefit of vehicle usage with the Expenditure report on a daily, weekly, or monthly basis.
Maintenance
The Maintenance schedule reminds you when you should have your vehicle serviced, such as an oil change or technical inspection. It can also serve as a reminder to take out insurance.
Easily Integrates
With Business
- APIs
The TeraTrack APIs will enable you to seamlessly integrate the real-time GPS ﬂeet tracking location and other data with your dispatching, payroll, mobile-resource management, CRM and/or ERP systems.
We will provide you with all the sample codes and documentations needed to get you up and running fast
OBJECT
CONTROL
Software offers Object control using commands which is extremely useful feature to control vehicles remotely. Turn on/off ignition and many other aspects. tracker
History and
Advanced
Reporting
Download and review reports in different formats: XLS, PDF, CSV and TXT. Reports contain various information categorized by date, such as GPS tracker name, driving hours, stopovers, distance traveled, fuel consumption and much more. Detailed, group and customized reports are available
Image Capturering
The Image gallery is used to store images taken by a camera connected to your tracking device or smartphone with the GPS Tracker mobile application installed
Instant
Notifications
Receive instant alerts about your tracked object, know when it enters or exits a geo-zone, when it is speeding and determine its stopovers. SOS alarms will turn on in case of any theft.

You will receive notiﬁcations on your iPhone, Android or Windows devices, via e-mail, mobile app or SMS
Fuel
Monitoring
Easily check the tank’s fuel level and fuel consumption along the route in real-time.

Receive instant notiﬁcation in case a fuel drop occurs, along with a relevant fuel report (with graph)
